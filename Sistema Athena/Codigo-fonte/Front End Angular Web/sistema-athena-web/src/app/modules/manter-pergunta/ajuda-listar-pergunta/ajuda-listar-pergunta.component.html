Orientações - Listar Pergunta

Operações

Listagem de todas as perguntas

Ao acessar a rota /manter-pergunta ou clicar no botão "Listar Pergunta", o usuário será redirecionado para uma tabela listando todas as perguntas disponíveis no sistema.

Atualização completa de uma pergunta

Uma vez escolhendo a pergunta desejada e clicando no botão "Atualizar Pergunta N", será exibido um modal que possibilitará a atualização completa daquela pergunta se nenhum campo for desabilitado.

No caso de perguntas de múltipla escolha, o nome das opções não podem conter dois traços seguidos "--", pois isso causará inconsistência na requisição.

Não é permitida a alteração de tipo de pergunta.

Atualização parcial de uma pergunta

Uma vez escolhendo a pergunta desejada e clicando no botão "Atualizar Pergunta N", será exibido um modal que possibilitará a atualização parcial daquela pergunta caso algum campo seja desabilitado pelo usuário por meio de deschecagem do respectivo checkbox.

No caso de perguntas de múltipla escolha, o nome das opções não podem conter dois traços seguidos "--", pois isso causará inconsistência na requisição.

Não é permitida a alteração de tipo de pergunta.

Exclusão de uma pergunta

Uma vez escolhendo a pergunta desejada e clicando no botão "Deletar Pergunta N", será exibido um modal que possibilitará a exclsuão daquela pergunta.

Campos

ID da Pergunta

Cada pergunta possui um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

ID da Prova - Nome da Prova

Cada prova possui tanto um nome quanto um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que uma certa pergunta está contida em uma prova.

ID da Disciplina - Nome da Disciplina

Cada prova possui tanto um nome quanto um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que uma certa pergunta pertence à Uma disciplina.

Enunciado

Toda a pergunta possui um enunciado, que será responsável pelo questionamento feito à pessoa que deve responder.

Tipo de Pergunta

Há dois tipos: Verdadeiro ou Falso; Múltipla Escolha. O tipo "Verdadeiro ou Falso" admite somente duas respostas(Verdadeiro ou Falso) e o tipo "Múltipla Escolha" admite cinco respostas, que são correspondentes ao enunciado. Das cinco, somente uma é verdadeira.

Resposta

Se o tipo da pergunta for "Verdadeiro ou Falso", então a resposta deve conter somente um caracter (1 para verdadeiro ou 0 para falso).

Já se for "Múltipla Escolha", então a resposta deve conter cinco caracteres, sendo 0 os caracteres correspondentes às alternativas falsas e 1 correspondente à alternativa verdadeira.

Exemplo: 00100, a terceira alternativa é a verdadeira.
