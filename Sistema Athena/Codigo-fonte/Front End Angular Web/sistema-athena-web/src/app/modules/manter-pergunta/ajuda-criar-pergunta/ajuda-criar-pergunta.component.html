Orientações - Criar Pergunta

Operações

Criar uma pergunta

Com acesso aos campos, o usuário pode preencher os campos e gerar uma pergunta.

No caso de perguntas de múltipla escolha, o nome das opções não podem conter dois traços seguidos "--", pois isso causará inconsistência na requisição.

Campos

ID da Pergunta

Cada pergunta possui um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

ID da Prova - Nome da Prova

Cada prova possui tanto um nome quanto um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que uma certa pergunta está contida em uma prova. O usuário deve escolher uma prova já cadastrada no sistema.

ID da Disciplina - Nome da Disciplina

Cada prova possui tanto um nome quanto um número identificador único associado à ela. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que uma certa pergunta pertence à Uma disciplina.. O usuário deve escolher uma disciplina já cadastrada no sistema.

Enunciado

Toda a pergunta possui um enunciado, que será responsável pelo questionamento feito à pessoa que deve responder.

Tipo de Pergunta

Há somente dois tipos: Verdadeiro ou Falso; Múltipla Escolha. O tipo "Verdadeiro ou Falso" admite somente duas respostas(Verdadeiro ou Falso) e o tipo "Múltipla Escolha" admite cinco respostas, que são correspondentes ao enunciado. Das cinco, somente uma é verdadeira.

Resposta

Se o tipo da pergunta for "Verdadeiro ou Falso", então a resposta deve conter somente um caracter (1 para verdadeiro ou 0 para falso).

Já se for "Múltipla Escolha", então a resposta deve conter cinco caracteres, sendo 0 os caracteres correspondentes às alternativas falsas e 1 correspondente à alternativa verdadeira.

Exemplo de respota de pergunta de múltipla escolha: 00100. A terceira alternativa é a verdadeira, as outras são falsas.
