Operações

Criar um Processador

Com acesso aos campos, o usuário pode preencher os campos e gerar um processador que ficará armazenado no sistema e será apresentado na listagem de computadores uma vez criado o processador.

Campos

Nome do Processador

Todo o processador possui um nome.

Marca do Processador

Todo o processador possui uma marca. Essa mesma marca pode ser uma distribuidora ou uma fabricante.

Quantidade de núcleos do Processador

Todo o processador possui uma certa quantidade de núcleos.

O núcleo do processado (Core) é uma unidade de processamento independente conectada à Unidade Central de Processamento (CPU). Atualmente, há modelos de chips que vão de dois até trinta e dois núcleos. Essencialmente, um núcleo pode ser uma CPU inteira.

Quantidade de threads do Processador

Todo o processador possui uma certa quantidade de threads.

Threads são entidades escalonadas para executarem na CPU. As mesmas threads concorrerão pelo processador juntamente com mais threads que tiverem no programa, ou concorrerá apenas com o fluxo do programa.

Preço do Processador

Todo o processador possui um preço, em reais.
