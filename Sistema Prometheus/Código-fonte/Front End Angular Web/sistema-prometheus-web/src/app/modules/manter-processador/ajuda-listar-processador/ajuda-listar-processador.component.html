Orientações - Listar Processador

Operações

Listagem de todas os Processadores

Ao acessar a rota /manter-processador ou clicar no botão "Listar Processador", o usuário será redirecionado para uma tabela listando todos os processadores disponíveis no sistema.

Atualização completa de um processador

Uma vez escolhendo o processador desejado e clicando no botão "Atualizar Processador N", será exibido um modal que possibilitará a atualização completa daquele processador se nenhum campo for desabilitado.

Atualização parcial de uma Processador

Uma vez escolhendo o processador desejado e clicando no botão "Atualizar Processador N", será exibido um modal que possibilitará a atualização parcial daquele processador caso algum campo seja desabilitado pelo usuário por meio de deschecagem do respectivo checkbox.

Exclusão de uma Processador

Uma vez escolhendo o processador desejado e clicando no botão "Deletar Processador N", será exibido um modal que possibilitará a exclsuão daquele processador.

Campos

ID da Processador

Cada processador possui um número de identificação único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

Nome do Processador

Todo o processador possui um nome.

Marca do Processador

Todo o processador possui uma marca. Essa mesma marca pode ser uma distribuidora ou uma fabricante.

Quantidade de núcleos do Processador

Todo o processador possui uma certa quantidade de núcleos.

O núcleo do processado (Core) é uma unidade de processamento independente conectada à Unidade Central de Processamento (CPU). Atualmente, há modelos de chips que vão de dois até trinta e dois núcleos. Essencialmente, um núcleo pode ser uma CPU inteira.

Quantidade de threads do Processador

Todo o processador possui uma certa quantidade de threads.

Threads são entidades escalonadas para executarem na CPU. As mesmas threads concorrerão pelo processador juntamente com mais threads que tiverem no programa, ou concorrerá apenas com o fluxo do programa.

Preço do Processador

Todo o processador possui um preço, em reais.
