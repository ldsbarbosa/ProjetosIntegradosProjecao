Orientações - Listar Computador

Operações

Listagem de todos os Computadores

Ao acessar a rota /manter-computador ou clicar no botão "Listar Computador", o usuário será redirecionado para uma tabela listando todos os computadores disponíveis no sistema que estão associados à ele.

Os computadores disponíveis na listagem são todos os computadores do sistema associados ao usuário que está autenticado.

Atualização completa de um Computador

Uma vez escolhendo o computador desejado e clicando no botão "Atualizar Computador N", será exibido um modal que possibilitará a atualização completa daquele computador se nenhum campo for desabilitado.

Atualização parcial de um Computador

Uma vez escolhendo o computador desejado e clicando no botão "Atualizar Computador N", será exibido um modal que possibilitará a atualização parcial daquele computador caso algum campo seja desabilitado pelo usuário por meio de deschecagem do respectivo checkbox.

Exclusão de um Computador

Uma vez escolhendo o computador desejado e clicando no botão "Deletar Computador N", será exibido um modal que possibilitará a exclsuão daquela Computador.

Campos

ID do Computador

Cada computador possui um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

ID do Gabinete - Nome do Gabinete

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.

ID da Fonte de Energia - Nome da Fonte de Energia

Cada fonte de energia possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.

ID do Armazenamento - Nome do Armazenamento

Cada armazenamento possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um armazenamento (SSD ou HD).

ID da Placa Mãe - Nome da Placa Mãe

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.

ID do Processador - Nome do Processador

Cada processador possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um processador.
