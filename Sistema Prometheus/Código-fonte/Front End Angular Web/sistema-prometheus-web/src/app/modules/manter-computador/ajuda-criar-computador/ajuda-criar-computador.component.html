Operações

Criar um Computador

Com acesso aos campos, o usuário pode preencher os campos e gerar um computador, que será apresentado na listagem de computadores uma vez criado o computador.

É importante a visualização da tabela "Listar Computadores", pois os números identificadores que podem ser inseridos devem ser feitos com base nos que já estão dispostos

Campos

ID do Computador

Cada computador possui um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

ID do Gabinete - Nome do Gabinete

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.

ID da Fonte de Energia - Nome da Fonte de Energia

Cada fonte de energia possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui uma fonte de energia.

ID do Armazenamento - Nome do Armazenamento

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui algum tipo de armazenamento, seja ele SSD ou HD.

ID da Placa Mãe - Nome da Placa Mãe

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui uma placa mãe.

ID do Processador - Nome do Processador

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.

ID da Memória RAM - Nome da Memória RAM - Quantidade de Pentes de Memória RAM

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui, pelo menos, um pente de memória RAM.

Compatibilidade

As peças que foram associadas à um computador podem ou não ser compatíveis entre si.

Preço de Montagem da Máquina

Uma vez de posse de todas as peças necessárias, a loja associará ao preço da máquina também o preço de montá-la, já que devem ser investido em pasta térmica, organização de cabos, etc.
