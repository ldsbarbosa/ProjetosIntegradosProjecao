Orientações - Associar Computador

Operações

Associação de Computadores

Ao acessar a rota /manter-computador e clicar no botão "Associar Computador", o usuário será redirecionado para uma tabela listando todas os computadores disponíveis no sistema para que possa associar um desses computadores à si, o disponibilizando para criações, alterações e exclusões.

Os computadores disponíveis na listagem são todos os computadores do sistema.

Campos

ID do Computador

Cada computador possui um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas.

ID do Gabinete - Nome do Gabinete

Cada gabinete possui tanto um nome quanto um número identificador único associado à ele. É importante o conhecimento deste número para que as operações de atualização e exclusão sejam realizadas, ja que um certo computador possui um gabinete.
